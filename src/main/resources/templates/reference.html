Reference of usage(Lviv - croissantDTOS - bot)

Common commands:

"menu" - menu of croissantDTOS

"navi" - navigation menu(only for Facebook messenger`s bot)

"courier_request" - request to admins on userCourier status(only for Facebook messenger`s bot)

"courier_actions" - userCourier actions(only for couriers)(only for Facebook messenger`s bot)

"personal_request" - request to admins on personal(only for Facebook messenger`s bot) status

"admin_request" - request to admins on admin status(only for Facebook messenger`s bot)

"abort" - abort any function(only for Facebook messenger`s bot)

Reference of usage(Lviv - croissants - bot)

Common commands:

"menu" - menu of croissants

"navi" - navigation menu

"courier_request" - request to admins on courier status

"courier_actions" - courier actions(only for couriers)

"personal_request" - request to admins on personal status

"admin_request" - request to admins on admin status

"abort" - abort any function
